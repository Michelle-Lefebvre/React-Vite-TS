# React-Vite-TS

React app built using Vite and TypeScript
To start:

##Install
Create App:
npm create vite@4.1.0

open app:
cd React-Vite-TS // app name

install dependencies:
npm install

open project in browser:
npm run dev

Bootstrap:
npm i bootstrap@5.2.3

##VS Code tricks
CMD + D // to select multiple text
